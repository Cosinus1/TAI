# Urban Mobility Analysis - Client

Angular frontend for urban mobility analysis system.

## Features

- Interactive maps for mobility visualization
- Real-time data dashboards
- Heatmap and flow analysis
- User-friendly interface for mobility insights

## Setup

This frontend will be implemented using Angular framework.

Faire un `npm ci` pour avoir `node_modules`

## Project Structure

- `src/` - Source code
  - `app/` - Angular application
  - `assets/` - Static assets
  - `environments/` - Environment configurations
- `public/` - Public assets
- `angular.json` - Angular configuration
